# 💪 Fitness Gym Website

A modern, responsive promotional website for a fitness brand, designed to attract new members by showcasing services, features, and providing a contact interface.

🔗 https://fitness-project-one.vercel.app/

## 🎨 Figma Design
🔗 https://www.figma.com/design/B5kMEjLTCBEqaUfLlY3oO1/FitnussApp?node-id=0-1&t=kpL8FeS4puiso64m-1


## 🧠 Project Overview

This project represents a promotional website for a gym, including:

- Hero Section
- About Section
- Services / Features Section
- Contact Section with form validation
- Modern UI with animations


## ⚙️ Tech Stack

- React (Frontend Library)
- Vite (Build Tool)
- Tailwind CSS (Styling)
- Framer Motion (Animations)
- Lucide React (Icons)


## ✨ Features

- ✅ Fully responsive design (Mobile, Tablet, Desktop)
- ✅ Modern UI/UX design based on Figma
- ✅ Service filtering & search functionality
- ✅ Smooth animations using Framer Motion
- ✅ Contact form with basic validation
- ✅ Clean component-based architecture

## 🛠️ Installation & Setup

Follow these steps to run the project locally:

bash
# Clone the repository
git clone https://github.com/SudilThathsara/fitness.git

# Navigate into the project folder
cd fitness

# Install dependencies
npm install

# Run development server
npm run dev
